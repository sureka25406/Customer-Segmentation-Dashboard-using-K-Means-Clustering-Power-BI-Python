# Customer Segmentation Dashboard using K-Means Clustering

## Project Overview

This project focuses on customer segmentation using the K-Means Clustering algorithm and visualizing the results through an interactive Power BI dashboard.

The goal is to identify distinct customer groups based on Annual Income and Spending Score, enabling businesses to make data-driven marketing and customer engagement decisions.

---

## Problem Statement

Businesses often struggle to understand different customer behaviors.

By applying machine learning techniques, customers can be segmented into meaningful groups for:

- Targeted Marketing
- Personalized Promotions
- Customer Retention
- Revenue Optimization

## Machine Learning Model

### K-Means Clustering

K-Means is an unsupervised machine learning algorithm used to group similar customers into clusters.

Features Used:

- Annual Income
- Spending Score

Output:

- Cluster Labels
- Customer Segments
